# Dashboard ECAP Store

## Description
Ce projet est un dashboard interactif réalisé avec Dash (Python).

Il permet d'analyser les ventes d'un magasin à travers plusieurs indicateurs et visualisations.

## Fonctionnalités
- Indicateurs KPI (chiffre d'affaires et nombre de ventes)
- Graphique d’évolution du chiffre d’affaires
- Diagramme des 10 meilleures ventes
- Tableau interactif des 100 dernières transactions
- Filtre par zone géographique

## Technologies utilisées
- Python
- Dash
- Plotly
- Pandas
